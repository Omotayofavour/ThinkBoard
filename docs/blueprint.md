# **App Name**: ThinkBoard

## Core Features:

- Idea Creation: Users can input and save new ideas with a title and description.
- Idea Display: Display all created ideas in an organized list or card-based layout.
- Tagging and Categorization: Users can add relevant tags to their ideas for easy organization and filtering.
- Search and Filter: Allow users to search through ideas by keywords or filter by tags.
- AI Idea Expander: Utilize a generative AI tool to expand on a brief idea, offering detailed suggestions or related concepts.
- Persistence: Store user-generated ideas persistently so they are available across sessions.

## Style Guidelines:

- The application centers on creativity and focused thinking, so a calm yet vibrant palette is chosen. Primary color: '#33A7C1' (a soft, inspiring blue-cyan). This color provides a refreshing focal point without being overpowering.
- Background color: '#F2F6F7'. This very light, desaturated blue-cyan creates an open and bright workspace that minimizes visual clutter and enhances focus.
- Accent color: '#3CC07E' (a harmonious green-turquoise). This analogous color provides visual interest and is used for calls to action or key highlight elements, ensuring good contrast and readability.
- Body and headline font: 'Inter' (sans-serif). Chosen for its clean, modern, and objective appearance, suitable for clear presentation of ideas and readable for both short titles and longer descriptions.
- Simple, crisp, and functional vector-based icons will be used to denote actions like 'add idea', 'search', or 'tag', maintaining a clean and intuitive interface.
- A minimalist and responsive layout is preferred, ensuring a clean and distraction-free environment that adapts well to various screen sizes. The focus is on content-first presentation.
- Subtle, non-disruptive animations such as slight fade-ins or expansions on hover for cards will be employed to provide fluid user feedback and enhance interaction without slowing down the experience.
